How does it work?


The Web offers different websites that provide the same type of data. For example, YouTube, Dailymotion and Vimeo all provide videos, playlists, users, etc. For us to be able to access and retrieve these website’s data in a reliable way, we need to “learn and understand” each website API (How many API endpoints/HTTP requests. We need to cal to get the videos from each site, and what are returned fields and how to access them, etc).

Users should be able to interact with websites that offers the same data in the same way, without having to know how each website implement and describe this data. To be able to do so, we need to connect the description of each website API to a common description (schema.org). Where we can use VideoObject schema to access videos in YouTube, Vimeo, Dailymotion, or any other video website!

Shapir allows users to connect the description of the website API to Schema.org vocabulary.
